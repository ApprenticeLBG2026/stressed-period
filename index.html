💼 Stressed Period Identification for IRC

Automated Credit Risk Calibration for Internal Model Method

Overview

This project automates the identification of credit-stressed periods to support Internal Model Method (IMM) calibration for IRC (Incremental Risk Charge).

Under CRR Article 284(4), banks using internal models must calibrate stressed inputs using an appropriate historical stress period. This project replaces a manual, spreadsheet-based process with a reproducible Python pipeline.

🔒 Regulatory Framework: CRR Article 284(4) | Internal Model Method (IMM) | Incremental Risk Charge (IRC)

Business Context

Banks using internal models for capital calculations must:

  • Identify and calibrate stressed periods from historical data
  • Use appropriate PD (Probability of Default) time series
  • Incorporate exposure-weighted EAD (Exposure at Default) data
  • Ensure reproducibility and auditability of calculations
  • Comply with regulatory validation requirements

Data Processing Pipeline

Input Data Sources

  • CCS Data: Credit conditions survey data (Q1 2026 annex - Corporate)
  • Yield Curves: GLC nominal daily spot curves from 2005 to present
  • Default Rates: Medium and Large PNFC default rate changes by quarter

Processing Steps

  • Extract 5-year yield curve data from multiple historical datasets
  • Aggregate yield data by quarter (Q) through resampling
  • Merge default rate changes with 5-year yields on common date labels
  • Calculate 4-quarter rolling averages for smoothing
  • Detect missing data and validate time series alignment

Implementation Code

Core data processing and stressed period identification:

Python - Credit Risk Data Pipeline (analysis/script.py)
import pandas as pd # Load credit conditions survey and yield curve data ccs_data = pd.read_excel(r"ccs-2026-q1-annex.xlsx", sheet_name="Corporate annex", header=None) yield_data_2005_2015 = pd.read_excel(r"GLC Nominal daily data_2005 to 2015.xlsx", sheet_name="4. spot curve", header=None) yield_data_2016_2024 = pd.read_excel(r"GLC Nominal daily data_2016 to 2024.xlsx", sheet_name="4. spot curve", header=None) yield_data_2025_present = pd.read_excel(r"GLC Nominal daily data_2025 to present.xlsx", sheet_name="4. spot curve", header=None) # Extract PNFC default rate changes by size category years = ccs_data.iloc[15, 2:] quarters = ccs_data.iloc[16, 2:] medium_pnfc = ccs_data.iloc[83, 2:] large_pnfc = ccs_data.iloc[86, 2:] years = years.ffill() labels = quarters.astype(str) + " " + years.astype(int).astype(str) # Create default rate change dataframe default_rate_change = pd.DataFrame({ "date": labels, "medium pnfc default rate change": medium_pnfc.values, "large pnfc default rate change": large_pnfc.values }).dropna().reset_index(drop=True) def extract_5y(df): """Extract 5-year yield from spot curve data""" out = pd.DataFrame({ "date": df.iloc[5:, 0], # column A, from row 6 "5y": df.iloc[5:, 10] # column K, from row 6 }) out["date"] = pd.to_datetime(out["date"], errors="coerce") return out.dropna() # Consolidate yield curves from multiple periods yield_5y_all = ( pd.concat([extract_5y(yield_data_2005_2015), extract_5y(yield_data_2016_2024), extract_5y(yield_data_2025_present)], ignore_index=True) .sort_values("date") .reset_index(drop=True) ) # Resample to quarterly frequency and align with default rates yield_5y_q = ( yield_5y_all.set_index("date")["5y"] .resample("Q") .mean() .reset_index() ) yield_5y_q["date"] = ("Q" + yield_5y_q["date"].dt.quarter.astype(str) + " " + yield_5y_q["date"].dt.year.astype(str)) # Merge and validate data alignment check_q = default_rate_change.merge(yield_5y_q, on="date", how="left") missing = check_q[check_q["5y"].isna()] # Calculate 4-quarter rolling averages for stress period smoothing check_q["medium_pnfc_ma4"] = ( check_q["medium pnfc default rate change"] .rolling(window=4, min_periods=4) .mean() ) check_q["large_pnfc_ma4"] = ( check_q["large pnfc default rate change"] .rolling(window=4, min_periods=4) .mean() ) check_q["5y_yield_ma4"] = ( check_q["5y"] .rolling(window=4, min_periods=4) .mean() ) # Export consolidated timeseries check_q.to_excel(r"ccs_and_yield_timeseries.xlsx", index=False)
